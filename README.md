# Анализ пролонгаций договоров 2023

## Описание проекта
Аналитический проект по расчету коэффициентов пролонгации договоров с клиентами для отдела сопровождения клиентов за 2023 год.

## Структура проекта
```
prolongation-analysis-2023/
├── data/                   # Исходные данные
│   ├── financial_data.csv  # Финансовые данные по проектам
│   └── prolongations.csv   # Данные о последних месяцах реализации проектов
├── notebooks/              # Jupyter notebooks с анализом
├── output/                 # Результаты анализа (отчеты, графики)
└── README.md
```

## Коэффициенты пролонгации

### K1 (пролонгация в первый месяц)
Отношение суммы отгрузки проектов, пролонгированных в первый месяц после завершения, к сумме отгрузки последнего месяца реализации всех завершившихся в прошлом месяце проектов.

### K2 (пролонгация во второй месяц)
Отношение суммы отгрузки проектов, пролонгированных во второй месяц, к сумме отгрузки последнего месяца проектов, не пролонгированных в первый.

## Результаты
- Коэффициенты для каждого менеджера (помесячно и за год)
- Коэффициенты по отделу в целом
- Аналитический отчет в Excel

## Использование
1. Установить зависимости: `pip install -r requirements.txt`
2. Открыть notebook: `jupyter notebook notebooks/prolongation_analysis.ipynb`
3. Запустить все ячейки для генерации отчета
